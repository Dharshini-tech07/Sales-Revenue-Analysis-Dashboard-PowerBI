# 📊 Sales & Revenue Analysis Dashboard using Power BI

## 📌 Project Overview
This project is an interactive Sales & Revenue Analysis Dashboard developed using Microsoft Power BI. It helps analyze sales performance through KPIs, charts, and interactive filters.

## 🚀 Features
- Total Sales KPI
- Total Profit KPI
- Total Quantity KPI
- Sales by Category
- Monthly Sales Trend
- Sales by Region
- Top Products Analysis
- Interactive Slicers (Category & Region)

## 🛠️ Tools Used
- Microsoft Power BI
- Microsoft Excel
